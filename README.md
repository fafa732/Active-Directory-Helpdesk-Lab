# Active Directory Help Desk Home Lab

## Project Overview
This project documents the step-by-step setup of a Windows Server Active Directory environment designed to simulate a real-world help desk scenario. 

The lab demonstrates how users, computers, policies, and support tasks are managed in a domain environment.

---

## Objectives
- Install and configure Windows Server
- Set up Active Directory Domain Services (AD DS)
- Create and manage users and groups
- Organize resources using Organizational Units (OUs)
- Join client machines to the domain
- Apply and test Group Policy
- Perform real help desk tasks (password reset, account unlock, etc.)
- Prepare for integration with ticketing and MDM solutions

---

## Technologies Used
- Windows Server 2022
- Active Directory Domain Services (AD DS)
- DNS
- Group Policy Management
- Windows 10 
- VirtualBox 
- PowerShell / Command Prompt
